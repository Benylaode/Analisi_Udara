# Analisi Udara Daschboard

## Setup Environment

### Shell/Terminal
```sh
python -m venv myenv
python myenv/bin/activate
pip install -r requirements.txt
mkdir proyek_analisis_data
cd proyek_analisis_data
clone "url_repo"
```

## Menjalankan Aplikasi
```sh
cd dashboard
streamlit run dasbor.py
```
